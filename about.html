About

ClamGUI® is an open source front-end (Graphical User Interface) for ClamAV. ClamGUI allows easier use, more control, and quarantining.

Features

  • Graphical window for ClamAV
  • Quarantine file manager
  • Up-to-date with ClamAV antivirus signatures
  • All the features of ClamAV!

Planned Features

  • Mail scanning and management
